Logfile Generator script

Takes a few CSV files to build a bogus log file with email addresses  
- baby-names.csv - +25K names, using this for a first name  
- surnames.csv - +150K names, using this for the last name  
- too-500-urls.csv - using this for domain name  
- Human-to-HumanActionableRequestsDataset.csv - using this of for a request type and a message. Messages are very conversational  

Geared for Linux systems (uses epoch time)
